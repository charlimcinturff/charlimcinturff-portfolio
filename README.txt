# Charli McInturff — Portfolio Website

This is a simple, clean, recruiter-friendly portfolio site (HTML + CSS).

## Files
- `index.html` — main page
- `styles.css` — styling

## How to run locally
1. Download the folder.
2. Double-click `index.html` to open in your browser.

## Publish for free (GitHub Pages)
1. Create a repo named `charli-portfolio` (or anything).
2. Upload `index.html` + `styles.css`.
3. In GitHub: Settings → Pages → Deploy from branch → `main` → `/root`
4. Your site will be live in ~1–2 minutes.

## Customize
- Add project links (GitHub, PDF write-ups, videos)
- Add images in an `assets/` folder and reference them in `index.html`


## Add your LinkedIn profile photo
- Save your headshot as `assets/profile.jpg`
- Then edit `index.html` and change the avatar src to `assets/profile.jpg` (or keep the placeholder SVG).
